Bramble.mesh — Function
mesh(Ω::Domain, npts::Int, unif::Bool)

Returns a Mesh1D based on Domain Ω and npts points with uniform spacing if unif is true (otherwise, the points are randomly generated on the domain).

For future reference, we denote the npts entries of vector pts as

\[x_i, \, i=1,\dots,N.\]

Example

julia> I = interval(0,1); Ωₕ = mesh(domain(I), 10, true)
1D Mesh
nPoints: 10
Markers: Dirichlet
source
mesh(Ω::Domain, npts::NTuple{D,Int}, unif::NTuple{D,Bool})

Returns a MeshnD ($1 \leq n \leq 3$) from the Domain Ω. The number of points for each coordinate projection mesh are given in the tuple npts. The distribution of points on the submeshes are encoded in the tuple unif. The Domain markers are translated to markers as for Mesh1D.

For future reference, the mesh points are denoted as

  • 2D mesh, with npts = ($N_x$, $N_y$)

\[(x_i,y_j), \, i=1,\dots,N_x, \, j=1,\dots,N_y\]

  • 3D mesh, with npts = ($N_x$, $N_y$, $N_z$)

\[(x_i,y_j,z_l), \, i=1,\dots,N_x, \, j=1,\dots,N_y, \, l=1,\dots,N_z.\]

Example

julia> X = Domain(Interval(0,1) × Interval(4,5)); Ωₕ = Mesh(X, (10, 15), (true, false))
2D Mesh
nPoints: 150
Markers: ["Dirichlet"]

Submeshes:
  x direction | nPoints: 10
  y direction | nPoints: 15

source
hₘₐₓ(Ωₕ::MeshnD)

Returns the maximum diagonal of mesh Ωₕ

  • 2D mesh

\[\max_{i,j} \Vert (h_{x,i}, h_{y,j}) \Vert_2\]

  • 3D mesh

\[\max_{i,j,l} \Vert (h_{x,i}, h_{y,j}, h_{z,l}) \Vert_2\]
